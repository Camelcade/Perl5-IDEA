<p style="padding-bottom: 10px;">There is no builtin <a href="psi_element://perlfunc%2Fimport+LIST"><code>import</code></a> function.  It is just an
ordinary method (subroutine) defined (or inherited) by modules that wish
to export names to another module.  The
<a href="psi_element://perlfunc%2Fuse+Module+VERSION+LIST"><code>use</code></a> function calls the
<a href="psi_element://perlfunc%2Fimport+LIST"><code>import</code></a> method for the package used.  See also
<a href="psi_element://perlfunc%2Fuse+Module+VERSION+LIST"><code>use</code></a>, <a href="psi_element://perlmod">perlmod</a>, and <a href="psi_element://Exporter" style="color:red">Exporter</a>.</p></dd></dl>

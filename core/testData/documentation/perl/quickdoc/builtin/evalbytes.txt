<dt style="padding-bottom:4px;font-weight:bold;">evalbytes</dt><dd style="padding-top:6px;">
<p style="padding-bottom: 10px;">This function is like <a href="psi_element://pod%3A%3Aperlfunc%2Feval+EXPR">C<eval></a> with a string argument,
except it always parses its argument, or <a href="psi_element://perlvar%2F%24_">C<$_></a> if EXPR is
omitted, as a string of bytes.  A string containing characters whose
ordinal value exceeds 255 results in an error.  Source filters activated
within the evaluated code apply to the code itself.</p>
<p style="padding-bottom: 10px;"><a href="psi_element://pod%3A%3Aperlfunc%2Fevalbytes+EXPR">C<evalbytes></a> is available only if the
<a href="psi_element://feature%2FThe+%27unicode_eval%27+and+%27evalbytes%27+features" style="color:red">C<"evalbytes"> feature</a>
is enabled or if it is prefixed with <code>CORE::</code>.  The
<a href="psi_element://feature%2FThe+%27unicode_eval%27+and+%27evalbytes%27+features" style="color:red">C<"evalbytes"> feature</a>
is enabled automatically with a <code>use v5.16</code> (or higher) declaration in
the current scope.</p></dd></dl>

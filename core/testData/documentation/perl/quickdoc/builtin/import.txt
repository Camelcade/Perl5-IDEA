<p style="padding-bottom: 10px;">There is no builtin <a href="psi_element://pod%3A%3Aperlfunc%2Fimport+LIST">C<import></a> function.  It is just an
ordinary method (subroutine) defined (or inherited) by modules that wish
to export names to another module.  The
<a href="psi_element://pod%3A%3Aperlfunc%2Fuse+Module+VERSION+LIST">C<use></a> function calls the
<a href="psi_element://pod%3A%3Aperlfunc%2Fimport+LIST">C<import></a> method for the package used.  See also
<a href="psi_element://pod%3A%3Aperlfunc%2Fuse+Module+VERSION+LIST">C<use></a>, <a href="psi_element://perlmod">perlmod</a>, and <a href="psi_element://Exporter" style="color:red">Exporter</a>.</p></dd></dl>

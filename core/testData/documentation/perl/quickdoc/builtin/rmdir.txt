<dt style="padding-bottom:4px;font-weight:bold;">rmdir</dt><dd style="padding-top:6px;">
<p style="padding-bottom: 10px;">Deletes the directory specified by FILENAME if that directory is
empty.  If it succeeds it returns true; otherwise it returns false and
sets <a href="psi_element://perlvar%2F%24%21">C<$!></a> (errno).  If FILENAME is omitted, uses
<a href="psi_element://perlvar%2F%24_">C<$_></a>.</p>
<p style="padding-bottom: 10px;">To remove a directory tree recursively (<code>rm -rf</code> on Unix) look at
the <a href="psi_element://File%3A%3APath%2Frmtree%28+%24dir+%29" style="color:red">C<rmtree></a> function of the <a href="psi_element://File%3A%3APath" style="color:red">File::Path</a>
module.</p></dd></dl>

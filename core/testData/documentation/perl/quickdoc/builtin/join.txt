<p><a href="psi_element://perlfunc">perlfunc</a>: <a href="psi_element://perlfunc%2FDESCRIPTION">DESCRIPTION</a>: <a href="psi_element://perlfunc%2FAlphabetical+Listing+of+Perl+Functions">Alphabetical Listing of Perl Functions</a></p><dl><dt style="padding-bottom:4px;font-weight:bold;">join EXPR,LIST</dt><dd style="padding-top:6px;">
<p style="padding-bottom: 10px;">Joins the separate strings of LIST into a single string with fields
separated by the value of EXPR, and returns that new string.  Example:</p>
<div style="padding-bottom: 10px;"><pre><code>   my $rec = join(':', $login,$passwd,$uid,$gid,$gcos,$home,$shell);</code></pre></div>
<p style="padding-bottom: 10px;">Beware that unlike <a href="psi_element://perlfunc%2Fsplit+%2FPATTERN%2F%2CEXPR%2CLIMIT"><code>split</code></a>,
<a href="psi_element://perlfunc%2Fjoin+EXPR%2CLIST"><code>join</code></a> doesn't take a pattern as its first argument.
Compare <a href="psi_element://perlfunc%2Fsplit+%2FPATTERN%2F%2CEXPR%2CLIMIT"><code>split</code></a>.</p></dd></dl>

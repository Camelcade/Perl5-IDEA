<dt style="padding-bottom:4px;font-weight:bold;">study</dt><dd style="padding-top:6px;">
<p style="padding-bottom: 10px;">At this time, <code>study</code> does nothing. This may change in the future.</p>
<p style="padding-bottom: 10px;">Prior to Perl version 5.16, it would create an inverted index of all characters
that occurred in the given SCALAR (or <a href="psi_element://perlvar%2F%24_">C<$_></a> if unspecified). When
matching a pattern, the rarest character from the pattern would be looked up in
this index. Rarity was based on some static frequency tables constructed from
some C programs and English text.</p></dd></dl>

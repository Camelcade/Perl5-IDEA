<p><a href="psi_element://pod%3A%3Aperlvar">pod::perlvar</a>: <a href="psi_element://pod%3A%3Aperlvar%2FSPECIAL+VARIABLES">SPECIAL VARIABLES</a>: <a href="psi_element://pod%3A%3Aperlvar%2FVariables+related+to+the+interpreter+state">Variables related to the interpreter state</a></p><dl><dt style="padding-bottom:4px;font-weight:bold;">${^TAINT}</dt><dd style="padding-top:6px;"><p style="padding-bottom: 10px;">Reflects if taint mode is on or off.  1 for on (the program was run with
<B>-T</B>), 0 for off, -1 when only taint warnings are enabled (i.e. with
<B>-t</B> or <B>-TU</B>).</p>
<p style="padding-bottom: 10px;">This variable is read-only.</p>
<p style="padding-bottom: 10px;">This variable was added in Perl v5.8.0.</p></dd></dl>

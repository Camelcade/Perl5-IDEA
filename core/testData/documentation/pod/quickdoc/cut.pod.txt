<p><a href="psi_element://pod%3A%3Aperlpod">pod::perlpod</a>: <a href="psi_element://pod%3A%3Aperlpod%2FDESCRIPTION">DESCRIPTION</a>: <a href="psi_element://pod%3A%3Aperlpod%2FCommand+Paragraph">Command Paragraph</a></p><dl><dt style="padding-bottom:4px;font-weight:bold;"><code>=cut</code>
 </dt><dd style="padding-top:6px;"><p style="padding-bottom: 10px;">To end a Pod block, use a blank line,
then a line beginning with &quot;=cut&quot;, and a blank
line after it.  This lets Perl (and the Pod formatter) know that
this is where Perl code is resuming.  (The blank line before the &quot;=cut&quot;
is not technically necessary, but many older Pod processors require it.)</p></dd></dl>
